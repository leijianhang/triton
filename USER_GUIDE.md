# User Guide

Use the Search Symbol dialog to choose an A-share, US stock, or Hong Kong stock.

After selecting a symbol, the main chart loads K-line data from the backend market API. Historical bars are lazy-loaded as the chart is moved left.

Use the top toolbar for chart style, timeframe, indicators, candle patterns, chart patterns, auto Fibonacci, trends, heatmap, and GoNoGo.

Use the watchlist and scanner panels to monitor selected symbols across the supported markets.
